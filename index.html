URL Phishing Detection

This study developed a machine learning-based system for phishing URL detection using Random Forest, Logistic Regression, and Gradient Boosting models. Initial high F1 scores indicated potential overfitting due to dataset simplicity. Further testing with a synthetic dataset revealed performance drops in Random Forest and Gradient Boosting, while Logistic Regression showed better generalization. The results highlight the need for rigorous evaluation on unseen data to ensure model robustness and real-world reliability.
